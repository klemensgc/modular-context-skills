## Example 1: Reklamy rekrutacyjne (Ad Copy)

**Sesja:** 2026-02-02
**Typ:** Ad copy — 4 warianty IG/FB
**Audience:** Kandydatki na recepcjonistkę + menedżer operacyjny
**Język:** PL

### Output (4 warianty)

**Wariant A — Krótki (15s Stories):**
```
🔴 Szukamy Recepcjonistki / Opiekunki Pacjenta | Apolonia Dental Szczecin

Dołącz do drużyny top 1%.

✓ Praca z najnowszą technologią
✓ Rozwój, szkolenia i możliwość awansu
✓ Rodzinna atmosfera, 35 lat tradycji

📍 Szczecin Prawobrzeże | Pełny etat

Aplikuj teraz 👇 Link w bio
```

**Wariant C — Heritage (storytelling):**
```
🔴 500 lat tradycji. Teraz Twoja kolej.

Rodzina Gąsieniców buduje rzeczy, które trwają.

35 lat w stomatologii. Teraz szukamy osoby na recepcję...
→ Lubi kontakt z ludźmi
→ Chce rosnąć razem z nami

Rozwijamy się - u nas możesz awansować i zwiększać swoje zarobki.
```

**Wariant D — Role-specific (menedżer):**
```
🔴 Menedżer Operacyjny | Apolonia Dental

Zarządzaj rosnącym zespołem. Buduj z nami coś dużego.

Co robimy:
→ 4 kliniki w Szczecinie, kolejne w planach
→ Własne laboratorium i technologia AI (ReceptionOS)
→ Projekty o skali międzynarodowej
```

### Hormozi applied (latent — nie nazwany w copy):
- **EVP:** "drużyna top 1%", "Apolonia Academy", "możliwość awansu"
- **Dream Outcome:** "Realizuj swój pełen potencjał"
- **Effort:** "Chce pracować z technologią, nie z papierem"
- **Time:** "rozwijamy się" (implicit: awans szybko)

## Example 2: Email template (oferta pracy)

**Sesja:** 2026-02-13
**Typ:** Email template — formalna oferta współpracy
**Audience:** Violetta (kandydatka na menedżera)
**Język:** PL

### Output:

```
Temat: Grupa Apolonia — zaproszenie do współpracy / Menedżer Operacyjny

Dzień dobry Pani Violetto,

dziękujemy za rozmowę rekrutacyjną — zrobiła na nas bardzo dobre
wrażenie zarówno pod kątem doświadczenia w zarządzaniu klinikami,
jak i podejścia do pacjenta, zespołu i analityki.

Z przyjemnością informujemy, że chcielibyśmy zaprosić Panią
do współpracy w roli Menedżera Operacyjnego Grupy Apolonia.

Poniżej kluczowe warunki:
- Forma: umowa o współpracy B2B
- Rola: Menedżer Operacyjny
- Wynagrodzenie: 10 000 zł netto miesięcznie
- Wymiar: pełny etat (~160h/mc), elastyczne godziny
- Okres próbny: 1 miesiąc
- Start: 1 marca 2026 r.

Prosimy o informację zwrotną do 20 lutego.

Z poważaniem,
Klemens Gąsienica-Ciułacz
```

### Pattern — Email oferty:
1. Opening: personalized compliment (co nas przekonało)
2. Statement: zaproszenie do współpracy
3. Warunki: bullet points (forma, rola, kwota, start)
4. Zakres: 1 akapit summary
5. CTA: deadline odpowiedzi
6. Podpis: imię + firma + email

## Example 3: Ad Copy — Setup dla publikującego

Do każdego zestawu ad copy dołączaj **setup guide** dla osoby publikującej:

```markdown
## Setup dla {imię}

### Instagram:
1. Wrzuć filmik (15s na Stories/Reels, 60s na post)
2. Dodaj tekst z wybranego wariantu
3. Link w bio → Linktree → formularz

### Facebook:
1. Post z filmikiem + tekst
2. CTA button: "Aplikuj teraz" → link do formularza
3. Boost z targetowaniem:
   - Lokalizacja: {miasto} + 30km
   - Wiek: {range}
   - Zainteresowania: {lista}
```

## Copy Formulas (z sesji)

### Ad Copy Structure
```
[emoji + hook] — 1 zdanie
[Value proposition] — 2-3 zdania
[Bullet benefits] — 3-5 punktów (✓ lub →)
[Location + format] — 📍
[CTA] — Aplikuj 👇
[Hashtags] — 4-6 relevantnych
```

### Heritage Variant Formula
```
[Bold heritage statement] — "500 lat tradycji"
[Family/founder credibility] — "Rodzina X buduje..."
[Timeline proof] — "35 lat w stomatologii"
[Bridge to role] — "Teraz szukamy..."
[Benefits] — bullet points
```

### Benefit Language Pattern
```
Feature → Patient/User Outcome

BAD:  "Mamy CBCT i panoramę"
GOOD: "Oszczędzasz czas — nie wysyłamy na RTG"

BAD:  "Używamy ReceptionOS"
GOOD: "Pracujesz z technologią, nie z papierem"

BAD:  "Prowadzimy szkolenia"
GOOD: "Apolonia Academy — mentoring i ścieżka awansu"
```
